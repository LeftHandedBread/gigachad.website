Server Hosting with Toast Host

Overview

Toast Host is a web based server hosting solution powered by Pelican Panel. To obtain login credentials, contact Bread via Discord (lefthandedbread).

Toast Host provides a user-friendly interface for server management and configuration, and allows you to create and manage game servers, including Minecraft, Valheim, Factorio and much more. A full list of supported servers can be viewed here.

Servers

After contacting an administrator for login credentials and which game server you want to host, you can login to the server hosting panel at toast-host.com. Once you have logged in, you will see a list of your servers.

Each server opens with the console where you can input commands specific to that server.

You can upload and edit files in the "files" tab on the sidebar.
