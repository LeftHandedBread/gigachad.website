About

This server is a casual server for the giga-est of chads. A place to hang out in Minecraft while you work on your homework.

Minimum Requirements

Minecraft Java Edition 1.20.1 on any supported PC. Bedrock edition is not supported.

This server is not vanilla conpatible. Please use the modpack linked below.

Using Minecraft With Mods

Fear not, for it is not hard with the right launcher.

We recommend you download and familiarize yourself with Prism Launcher, as it is very versatile, lightweight, and stable. You may use another more user-friendly launcher if you like, such as the CurseForge Launcher, or the Default launcher. However, pack development is much simpler with Prism Launcher, so we use it by default.

Please ask in the Discord if you need help with setting up the modded launchers.
