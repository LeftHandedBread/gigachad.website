# Gigachad Network

A self-hosted services and game server network. Take back control of your digital life.

## About

The Gigachad Network is a collection of self-hosted services and game servers designed to provide privacy-focused alternatives to mainstream tech platforms. All services are ad-free, payment-free, and keep your data under your control.

## Services

- **Jellyfin** - Media streaming server (movies, TV shows, music)
- **Jellyseerr** - Content request system for Jellyfin
- **AudiobookShelf** - Audiobook and podcast streaming
- **NextCloud** - Personal cloud storage and file sync
- **Gigachad AI** - AI chat and productivity tools
- **VPN/WireGuard** - Secure network access
- **Vaultwarden** - Password manager (coming soon)
- **Navidrome** - Music streaming server
- **Toast Host** - Game server hosting platform

## Game Servers

- Minecraft Create Server
- Minecraft Vanilla Server (1.21.11, Java/Bedrock crossplay)
- Minecraft Abyssal Ascent
- Valheim Server
- Factorio Space Age
