Valheim

Modded • Immersive Mode

POV: You died in the plains and are trying to get your stuff back.

About

This server is a casual server for the giga-est of chads focusing on mostly QoL mods. The idea is that it puts less stress on the players that are new to modded, while giving older players a refreshed Valheim experience.

Minimum Requirements

Latest version of Valheim.

This server is not vanilla compatible. Please use the modpack (install instructions below).

Playing Valheim With Mods

Fear not, for it is not hard with the right launcher.

There are a few great launchers that make it easy to add mods, such as r2modman, which is what we use. Simply install the launcher, then follow the instructions as seen below:

  • Open R2modman
  • Select Valheim (Pressing star brings it to the front when you open the mod manager)
  • Select Steam
  • Select "Import" to import a profile
  • Select "Import new profile" then "From code"
  • Type the code 01906ba6-c795-a9ef-4c01-99c97ec1c03e into the text box
  • Create the profile
  • Wait for the mods to download
  • Press "Start Modded" to launch the game

Modpack code: 01906ba6-c795-a9ef-4c01-99c97ec1c03e

If you need help with setting up the modded launcher, or want to install manually, come ask in the Discord.
