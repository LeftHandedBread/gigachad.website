Recommended Launcher

We recommend you download and familiarize yourself with Prism Launcher, as it is very versatile, lightweight, and stable. Prism Launcher is a free, open-source Minecraft launcher that lets you manage multiple instances of the game, each with their own mods, settings, and versions.

Using Minecraft With Mods

Fear not, for it is not hard with the right launcher.

Importing the Pack into Prism Launcher

  1. Download the Prism Launcher Pack from the Downloads section below.
  2. Open Prism Launcher and click Add Instance at the top.
  3. Select Import on the left side of the dialog.
  4. Browse to the downloaded .zip file and select it.
  5. Click OK â€” Prism will set up the instance with all the required mods automatically.
  6. Double-click the new instance to launch the game and connect to the server.

Please ask in the Discord if you need help with setting up the modded launchers.
