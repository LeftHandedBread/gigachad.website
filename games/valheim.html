About

This server is a casual server for the giga-est of chads focusing on mostly QoL mods. The idea is that it puts less stress on the players that are new to modded, while giving older players a refreshed Valheim experience.

Playing Valheim With Mods

Fear not, for it is not hard with the right launcher. We use r2modman — install it, then follow the steps below.

  1. Open R2modman
  2. Select Valheim (press the star to pin it to the front)
  3. Select Steam
  4. Select "Import" to import a profile
  5. Select "Import new profile" then "From code"
  6. Enter the code: 019d1d7f-566f-e79a-7c41-f14739c82ab4
  7. Create the profile and wait for the mods to download
  8. Press "Start Modded" to launch the game

Modpack code: 019d1d7f-566f-e79a-7c41-f14739c82ab4

FAQ

Q: Are there any world modifiers?

A: Minimap is disabled, but we have the cartography table to navigate.

Q: Can I have the world seed?

A: No. We're playing Immersive mode, so that kind of defeats the point.

Q: What mods are installed on the server?

A: Here's the current mod list.

Mods

  • Gizmo (v1.15.0)
    Comfy version of Gizmo for easier building rotation controls.
  • PlantEasily (v2.1.0)
    Even crop spacing, bulk planting/harvesting, and optional auto-replant.
  • PlantEverything (v1.20.0)
    Lets the cultivator plant more flora like berries, mushrooms, and decorative plants.
  • LongshipUpgrades (v1.0.15)
    Upgrade longships with lanterns, sails, tent, shields, and expanded storage.
  • NoMap_Cartography_Map (v1.0.1)
    Enables map viewing at the cartography table during No Map play.
  • EpicLoot (v0.12.11)
    Adds loot drops, magic items, and enchanting.
  • Cooking (v1.2.2)
    Adds a cooking skill that boosts food-related stats and regeneration.
  • CookingAdditions (v1.3.1)
    Adds many new foods, ingredients, and additional cooking/processing stations.
  • ComfyQuickSlots (v1.9.0)
    Comfy fork of EquipmentAndQuickSlots with dedicated gear and quick-slot support.
  • ValheimArmory (v1.26.1)
    Expands weapon variety to support more playstyles across progression.
